# 优化React组件渲染   

## 使用大量的小组件  
`@observer`组件会追踪它们使用的所有值，并且当他们中任何一个改变时重新渲染。所以你的组件越小，它们需要重新渲染产生的变化则越小。  
这意味着用户界面的更多部分具备彼此独立渲染的可能性  

## 在专有组件中渲染列表  
这点在渲染大型数据集合时尤为重要。React在渲染大型数据集合时表现非常糟糕，因为协调器必须评估每个集合变化的集合所产生的组件。因此，建议使用专门的组件来映射集合并渲染这个组件，且不再渲染其他组件。
