# 这里是开发中的使用技巧

## VScode  
1. `ctrl+h`：查找并替换  
2. `/**`：Document this，添加注释  

## Git  
1. `git pull -p` :在本地删除远程已经删除的分支   
2. `q`，`git branch`后退出编辑模式  
 
## Github  
1. 推荐Chrome插件：`Octotree`、`Github Stars Manager`  
2. issue里“：”添加emoji 
3. issues里面 选中文字，按R键可以快速引用
4. 在github任意界面，按Shift + ? 显示当前可用快捷键
